# Agents of Change — National Agentic AI Bootcamp & Buildathon

Official website for OIST ACM-W's Agentic AI Bootcamp & Buildathon (10–11 July 2026).

## Stack
- Next.js (App Router) + TypeScript
- Tailwind CSS v4
- Framer Motion
- Lenis
- Lucide Icons

## Project Structure

```text
app/
components/
components/sections/
components/ui/
data/
lib/
public/
```

## Getting Started

```bash
npm install
npm run dev
```

Visit:

```
http://localhost:3000
```
